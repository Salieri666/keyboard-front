Использование приложения:

1. Управление учетными записями
 2. Управление упражнениями
 3. Выход из системы


1. Управление учетными записями

Для просмотра списка пользователей перейдите на вкладку "Пользователи".

Для добавления учетной записи пользователя нажмите на кнопку "Добавить".

Для изменения учетной записи пользователя нажмите на кнопку "Изменить" у соответствующей записи.

Введите логин и пароль изменяемой или добавляемой учетной записи и нажмите "Принять".

Для просмотра статистики пользователя нажмите на кнопку "Статистика" у соответствующего пользователя.

После этого откроется новая страница, содержащая статистику выбранного пользователя.

2. Управление упражнениями

Для добавления упражнения нажмите на кнопку "Добавить", после чего откроется диалоговое окно.

Введите желаемое название для упражнения.

Для загрузки текста упражнения из файла нажмите на кнопку "Загрузить из файла", после чего откроется диалоговое окно операционной системы для выбора файла.
Выберите желаемый файл в формате .txt и нажмите "Открыть". Текст упражнения отобразится в соответствующем поле.

Для автоматической генерации нажмите на кнопку "Сегенерировать", после чего уровень сложности и текст упражнения отобразятся в соответствующих полях.

Для добавления упражнения вручную: выберите в списке "Уровень сложности" желаемый уровень сложности и введите текст упражнения в соответствующее поле.

Для добавления составленного упражнения нажмите на кнопку "Принять".

Для измения упражнения нажмите на кнопку "Изменить" у упражнения в списке, после чего откроется диалоговое окно.

Введите желаемое или оставьте прежним название для упражнения.

Для загрузки текста упражнения из файла нажмите на кнопку "Загрузить из файла", после чего откроется диалоговое окно операционной системы для выбора файла.
Выберите желаемый файл в формате .txt и нажмите "Открыть". Текст упражнения отобразится в соответствующем поле.

Для автоматической генерации нажмите на кнопку "Сегенерировать", после чего уровень сложности и текст упражнения отобразятся в соответствующих полях.

Для измения упражнения вручную: выберите в списке "Уровень сложности" желаемый уровень сложности и введите текст упражнения в соответствующее поле.

Для измения составленного упражнения нажмите на кнопку "Принять".

Для измения настроек уровня сложности нажмите на кнопку "Уровни сложности", после чего откроется диалоговое окно.

Выберите в списке "Уровень сложности" необходимый уровень сложности. Введите желаемые настройки в соответствующие поля. По завершению редактирования нажмите "Принять" для сохранения изменений.

Для просмотра статистики упражнения нажмите на кнопку "Статистика" у соответствующего упражнения в списке, после чего откроется новая страница со статистикой выбранного пользователя.

3. Выход из системы

Для выхода из системы перейдите на вкладку "Профиль" и нажмите на кнопку "Выйти".

Использование приложения:

1. Прохождение тестового упражнения
 2. Тренировка
 3. Просмотр собственной статистики
 4. Выход из системы

1. Прохождение тестового упражнения

Для прохождения тестового упражнения необходимо нажать на панели вкладку "Тест".

На открытой странице будет отображено задание, клавиатура, результаты на текущий момент.
Для начала тестирования необходимо нажать на "Start".

Во время прохождения тестового задания необходимо нажимать на подсвеченную кнопку. Текущий результат можно увидеть над строкой задания.

Нажатием на кнопку "Toggle Keyboard" можно скрыть/отобразить виртуальную клавиатуру с подсветкой текущей клавиши.

По завершению выполнения упражнения отобразится конечный результат и укажется рекомендуемая сложность.

2. Тренировка

Для выполнения упражнений необходимо выбрать на панели вкладку "Тренировка".

На открытой странице будет отображен список доступных упражнений, по нажатию на список "Уровень сложности" можно отфильтровать упражнения.
По нажатию на кнопку "Выбрать" у соответствующего упражнения - откроется страница с заданием.

На открытой странице будет отображено задание, клавиатура, результаты на текущий момент.
Для начала необходимо нажать на "Start".

Во время прохождения задания необходимо нажимать на подсвеченную кнопку. Текущий результат можно увидеть над строкой задания.

Нажатием на кнопку "Toggle Keyboard" можно скрыть/отобразить виртуальную клавиатуру с подсветкой текущей клавиши.

По завершению выполнения упражнения отобразится конечный результат выполнения задания.

По нажатию на кнопку "Ok" откроется страница со списком упражнений.

3. Просмотр собственной статистики

Для просмотра собственной статистики необходимо на панели выбрать вкладку "Статистика".

Откроется страница содержащая статистику о пользователе на текущий момент.

4. Выход из системы

Для выхода из системы перейдите на вкладку "Профиль" и нажмите на кнопку "Выйти".
